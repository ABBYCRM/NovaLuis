NOVA Companion Integration

FluidVoice pairing

Keep speech recognition and microphone access on your Mac. FluidVoice sends only the resulting request to NOVA, where OpenClaw and Poolside Laguna execute it.

The PIN is sent only to NOVA over HTTPS. It is not saved in this page or included in the FluidVoice device token.

  1. Install FluidVoice on a supported Mac.

  3. Paste the Base URL, Model and API key shown above.
  4. Keep the API key in macOS Keychain. Do not paste the NVIDIA key into FluidVoice.
