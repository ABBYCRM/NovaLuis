## Purpose

The Composio Skills Catalog contains 832+ pre-built skill wrappers for the Composio SDK. These skills wrap common Composio tools (GitHub, Slack, Notion, Linear, Stripe, Resend, etc.) into agent-friendly skills.

## When to use

- Building agents that need external service integrations (GitHub, Slack, Notion, Linear, etc.)
- Using Composio as the tool execution layer
- When you need MCP-compatible tool wrappers

## Sub-skills

Each service has its own subdirectory with a dedicated SKILL.md:

- `ably-automation` — Ably real-time messaging automation
- `github-automation` — GitHub PR/issue/comment automation
- `slack-automation` — Slack messaging and channel operations
- `notion-automation` — Notion page/database operations
- `linear-automation` — Linear issue tracking automation
- `stripe-automation` — Stripe payment and subscription management
- `resend-automation` — Email automation via Resend
- (and 825 more)
