# OSINT Tools Collection

A curated set of open-source OSINT (Open-Source Intelligence) tools integrated into the ABBYCRM skills framework.

## Available Tools

| Tool | Description | Language | Stars |
|------|-------------|----------|-------|
| theHarvester | Email, subdomain & vhost harvester | Python | 19k+ |
| SpiderFoot | Automated OSINT reconnaissance | Python | 7k+ |
| Amass | In-depth subdomain enumeration | Go | 14k+ |
| Sublist3r | Subdomain enumeration | Python | 9k+ |
| Sherlock | Username hunt across social platforms | Python | 43k+ |
| WhatsMyName | Find accounts by username | Python | 2k+ |
| OSINT-Framework | Curated OSINT framework | Python | 3k+ |

## Usage

Each tool is a standalone script. See individual subdirectory SKILL.md files for detailed usage.

## Tools

### theHarvester
Email and subdomain discovery using public sources (Google, Bing, LinkedIn, etc.)

### SpiderFoot
Automated reconnaissance - maps attack surfaces and identifies exposures.

### Amass
DNS enumeration, network mapping, and attack surface discovery by OWASP.

### Sublist3r
Fast subdomain enumeration using search engines.

### Sherlock
Search for a username across 300+ social networks.

### WhatsMyName
Unified username discovery across web platforms.

### OSINT-Framework
Web-based OSINT framework with searchable database.
